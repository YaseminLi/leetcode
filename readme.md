### 集合、列表、数组
- 集合：一个或多个元素构成的一个整体，元素类型不一定相同，没有顺序。
- 列表（又称线性列表）的定义为：是一种数据项构成的有限序列，即按照一定的线性顺序，排列而成的数据项的集合。有顺序，长度可变，包括数组和链表。
- 数组：列表的实现方式之一。有索引，元素在内存中相邻。
